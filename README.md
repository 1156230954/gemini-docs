## 环境准备

先安装nodejs

再安装pnpm:
https://pnpm.io/zh/cli/install

参考：https://theme-hope.vuejs.press/zh/cookbook/tutorial/env.html#node-js


## 克隆代码仓库，并安装依赖：

```
pnpm install --frozen-lockfile
```

## 预览项目文档网站：

```
pnpm docs:dev
```
